# AlphaFund 品牌设计规范

## 🎨 Logo 设计

### 设计理念
```
    ●
    │
  ╱ ╲
 ╱   ╲
╱     ╲
│  A  │
│     │
╰─────╯
  ↗ ↖
```

- **字母A** - 代表Alpha（超额收益），基金投资的核心追求
- **神经网络节点** - 3个节点代表数据、智能、收益三位一体
- **渐变色彩** - 蓝色(#1E40AF)到青色(#06B6D4)的过渡

### 图标文件
| 文件 | 用途 |
|------|------|
| `icon_design.svg` | 设计源文件（完整版） |
| `launcher_icon.svg` | 启动器图标 |
| `icon_design_spec.json` | 设计规范文档 |

## 🎨 色彩系统

### 主色调
| 颜色 | 色值 | 用途 |
|------|------|------|
| 主色 | `#1E40AF` | 品牌主色、按钮、重要强调 |
| 主色浅 | `#3B82F6` | 悬浮状态、次要强调 |
| 主色深 | `#1E3A8A` | 深色模式主色 |
| 辅色 | `#06B6D4` | 次要强调、链接 |

### 语义色
| 颜色 | 色值 | 用途 |
|------|------|------|
| 上涨 | `#26A69A` | 正收益、价格上涨 |
| 下跌 | `#EF5350` | 负收益、价格下跌 |
| 警告 | `#FFA726` | 提醒、注意 |
| 成功 | `#26A69A` | 成功状态 |

### 渐变
```
┌─────────────────────────┐
│ #1E40AF → #3B82F6 → #06B6D4 │
└─────────────────────────┘
```

## 🔤 字体规范

### 品牌字体
- **中文**: Noto Sans SC（思源黑体）
- **英文/数字**: Noto Sans

### 字重
| 字重 | 用法 |
|------|------|
| Bold (700) | 标题、重要数据 |
| SemiBold (600) | 次标题 |
| Medium (500) | 正文强调 |
| Regular (400) | 正文 |

### 字阶
| 类型 | 大小 | 行高 | 用法 |
|------|------|------|------|
| Display | 57px | 64px | 超大数字 |
| Headline | 32px | 40px | 页面标题 |
| Title | 22px | 28px | 卡片标题 |
| Body | 16px | 24px | 正文内容 |
| Caption | 12px | 16px | 次要说明 |

## 📐 设计原则

### 圆角
- 卡片: 16px
- 按钮: 12px
- 输入框: 12px
- 弹窗: 20px

### 间距
- 页面边距: 16px
- 卡片内边距: 16px
- 元素间距: 12px
- 分组间距: 24px

### 阴影
- 浅阴影: 2dp / 5% opacity
- 中阴影: 4dp / 10% opacity
- 深阴影: 8dp / 15% opacity
